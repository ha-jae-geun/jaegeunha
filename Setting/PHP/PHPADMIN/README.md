# 설명
```java
이름 - 말그대로 컬럼의 이름입니다. 한글은 넣지 않는게 좋습니다. 

종류 - int(숫자) varchar(문자) date(시간) 등... 넣고싶은 정보의 자료형을 설정하는 부분입니다.

길이 - varchar 의 최대 길이는 4000자 숫자는 38 자 입니다. 

기본값 - default 값이라고도 하는데 값을 넣을때 만약 값이 없다면 사용자가 설정한 default값이 대신해서 들어가는 겁니다.

           원하지 않는 다면 '없음' 으로 두시면 됩니다.

데이터정렬방식 - 이것도 기본은 utf8_general_ci 입니다. 그냥 넘어가면 됩니다.


보기 - 음... 이건 정확 한건 모르겠습니다만 on update CURRENT_TIME 이라고 있는걸 보니 해당 레코드를 꺼낼때마다 동작을 하는 것 

       같습니다. 우리가 가끔 어플리케이션을 사용하다보면 '최근 접속 시간' 이란게 보이지 않습니까?? 그런 기술 같습니다.


Null - 중요한겁니다. 체크를 하지 않으면 NOT NULL(값을 넣을 때 해당 컬럼에 값이 없으면 에러발생) 

                       체크를 하면 NULL(값을 넣을 때 해당 컬럼에 값이 없어도 무관)

인덱스 - 제가 사용하는 건 PRIMARY 밖에 없습니다. 의미는 중복되는 값이 들어올때 에러를 발생시킨다는 의미 입니다.


A.I - Auto Increase 입니다. 알아서 숫자가 증가한다는 의미로 주로 게시판을 쓸때 사용하죠. 

      만약 A.I 가 체크되어 있는 컬럼이있다면 PRIMARY 또한 그 컬럼에 해야합니다. 안그러면 에러가 발생합니다.

ex) 이름에 PRIMARY 설정이 되어있는데 나이에 A.I 가 체크되어있으면 에러가 발생합니다.

     이름에 PRIMARY 를 취소하고 나이에 PRIMARY 와 A.I 를 같이 체크설정 해야 만들어집니다. 



설명은 컬럼의 역할을 적으라는 겁니다.
```
