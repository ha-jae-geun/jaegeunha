# 옥텟
```java
바이트(byte) 와 옥텟(octet) 에대해 알아보자.

8bit

바이트(byte) 또는 옥텟(octet)은 컴퓨터의 저장 단위 : bit 8개 를 일컷는다.
왜 같은 용어를 통해서 컴퓨터의 저장 단위를 이야기 하는걸까?

먼저 byte는 원래부터 8bit를 이야기 하지 않았다.

예전에는 4bit나 6bit를 byte라고 사용하는 컴퓨터가 있었다. 즉 컴퓨터마다 byte의 bit 기준이 달랐다.

그렇지만 현대에 와서는 8bit를 byte로 사용하는게 표준처럼 되었고, C언어에서도 8bit 이상을 byte로 사용하게 규정되었다.



octet는 처음부터 8bit를 칭하는 용어다.


위에서 이야기했던 것처럼 기존 컴퓨터들의 byte의 bit 기준이 다르다보니 
명확하게 8bit를 표현하기 위한 용어가 필요했고

그 이유에 의해서 생긴 용어이기 때문이다.

결론적으로 지금은 byte와 octet는 동일한 의미로 사용되고 있다.

추가적으로 8bit의 절반인 4bit는 니블(nibble) 이라는 용어로 사용한다.

※ 이 포스트에서 이야기하는 옥텟이란 용어는 컴퓨터에서 사용되는 옥텟(octet)를 설명 합니다. 
```
