- 충돌 저항성(Collision Resistance) : 같은 출력(h(x) = h(x′))을 갖는 임의

저항 해시 함수)이다. 이에 대한 안전성은 n/2 비트이다.

을 확률이 충분히 높아지는지를 묻는 문제이다. 얼핏 생각하기에는 생일
이 365~366가지이므로 임의의 두 사람의 생일이 같을 확률은 1/365~
1/366이고, 따라서 365명쯤은 모여야 생일이 같은 사람이 있을 것이라고
