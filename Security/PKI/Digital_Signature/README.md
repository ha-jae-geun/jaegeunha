[해설]
전자 서명의 특징
- 위조불가: 서명자만이 서명문을 생성할 수 있다. - 부인방지: 서명자는 서명 후에 사실을 부인할 수 없다. - 재사용 불가: 한번 서명한 서명문은 또 다른 문서에 사용할 수 없다. - 변경 불가: 내용 변경시 서명문 자체가 변경되어 변조 사실을 확인이
가능하다. - 서명자 인증: 서명자의 서명문은 서명자의 식별이 가능하다.
