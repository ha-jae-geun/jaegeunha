# 뷰
* 데이터베이스의 Select문을 저장한 OBJECT(쿼리문에서 Test처럼 사용)
- 가상테이블; 한개의 테이블에서 한 개만 생성
- 용도: 자주 검색하는 항목; 보안되는 항목
	* 보안성이 필요한 테이블의 요소를 다 공개하지 않고 뷰로 만들어달라고 요청
```java
 - MSSQL 뷰 수정방법

   ALTER VIEW [뷰명] AS
   SELECT [컬럼명] FROM [테이블명] WHERE [조건절]

 - MSSQL 뷰 삭제방법
   DROP VIEW [뷰명]

```
