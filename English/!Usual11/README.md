 * - 어린 아이들이 모여 서 있다가 “scramble!”라고 할 때??

 * “앞으로 좀 가줄래?” / “앞으로 당겨 가.”
 *  Can you move up?
 *  Can you scoot up?
 *  Can you scooch up?

 *  - I woke up so late this morning that I had to scramble to get dressed.
