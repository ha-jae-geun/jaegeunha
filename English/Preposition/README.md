사용
* 일단 adverb로서의 just가 활용되는 예가 중요해요. ad : 앞에 verb : 동사. ▶동사(형용사.다른 부사도 됨) 앞에서 동사를 꾸며주는건? "부사" 네. 부사로서 just의 활용은 크게 다섯가지로 요약됩니다.
