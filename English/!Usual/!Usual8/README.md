*   ~ 때문에 전화했어요.

* - dispute

*   돈을 내라고 하는 그림

* - statement
*   (특정 사항에 대해 명확히 공식적으로) 말하거나 쓴 것
* *크레딧 카드/ 은행과 관련된 문맥에서는 "내역서" 정도 되겠지요?

* - incorrect statement
*   (사실과) 맞지 않는 내역서
* ((incorrect : 수치, 철자가) 틀린 그림)

## <랜트카 빌릴 때 쓸 수 있는 영어 / 영어 공부 / 영어 회화>
