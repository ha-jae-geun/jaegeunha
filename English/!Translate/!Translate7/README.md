# [ 영어 대화가이드 #005 ] | 주말 끝나고 월요일에 나누는 인사

 * B: 괜찮았어. 별로 한건 없었어. 넌?
 * It was OK. I didn't do much. How about you?
 * A: 나도. 사실 이번 주를 위해 기운을 좀 아끼고 있긴 했어.
 * Me neither.
