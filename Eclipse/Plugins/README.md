```java
⎼ 는 바이트 코드 파일을 로 해주는 플러그인이다  jadclipse (.class) .java Decopiling . 
⎼ 확장자로 제공되는 라이브러리의 소스를 파악할 수 있다 class . 
⎼이클립스에서 로 제공되는 파일들의 소스를 확인 할 때는 단어를 선택하고 버튼을 클릭하면 Java api System F3 
단어에 대한 설명이 나오는데 이 설명이 소스 파일이다.
⎼소스 파일들을 보기 위해서는 제공받는 라이브러리에 있는 소스 파일들을 임포트 해야 하는데
```
