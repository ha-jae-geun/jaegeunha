# 배열과 캐시
* 배열은 데이터가 메모리공간에서 연속적으로 위치한다. 따라서, 캐시의 공간 지역성 원리에 의해 조회시에 높은 성능을 가질 확률이 높다.
* 그러나, 첫 할당시에 전체 메모리공간을 미리 할당해야 하며, 이 때문에 메모리 비효율적인 현상이 발견된다
