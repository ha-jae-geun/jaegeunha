# 라리브러리
```java
src/main/webapp/WEB-INF에 Folder를 하나 만들고 이름을 lib로 지정해줍니다.

오라클을 다운받아서 설치하셨으면 설치 폴더 경로

product/11.2.0/dbhome_1/jdbc/lib 에 ojdbc6 jar파일 있습니다.

복사를하시고 이클립스에서 만든 lib폴더에 붙여 넣어줍니다.
lib폴더에 ojdbc6.jar을 추가하셨으면 ojdbc6.jar 우클릭후 Build Path > Add to Build Path클릭.

Referenced Libraries에 ojdbc6.jar이 추가되었으면 성공한것입니다.


```
