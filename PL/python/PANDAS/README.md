## DataFrame 옵션
- DataFrame는 다음과 같은 옵션을 가지고 있다.
- ① data 옵션 : 실제 데이터의 값
- ② index 옵션 : 실제 데이터에 접근할 행의 이름
- ③ dtype 옵션 : 자료형
- ④ columns 옵션 : 실제 데이터에 접근할 열의 이름
