# 형용사
```java
밑줄 친 말의 품사가 같은 것으로만 묶은 것은? (2017지방9 B책형 문14)
① ㉠, ㉢, ㉣ ② ㉠, ㉣, ㉤ ③ ㉡, ㉢, ㉤ ④ ㉢, ㉣, ㉤
㉠흐드러지게(기본형: 흐드러지다): 「형용사」 매우 탐스럽거나 한창 성하다.≒ 흐무러지다 / 매우 흐뭇하거나 푸
지다. ◆ ‘흐드러지다’는 형용사로만 쓰인다. 
㉡찍은(기본형: 찍다): 「동사」 어떤 대상을 촬영기로 비추어 그 모양을 옮기다. 
◆ ‘찍다’는 동사로만 쓰인다. 
㉢설레는(기본형: 설레다): 「동사」 마음이 가라앉지 아니하고 들떠서 두근거리다. / 가만히 있지 아니하고 자꾸만 
움직이다. / 물 따위가 설설 끓거나 일렁거리다. ◆ ‘설레다’는 동사로만 쓰인다. 
㉣충만한(기본형: 충만하다): 「형용사」 한껏 차서 가득하다. ◆ ‘충만하다’는 형용사로만 쓰인다. 
㉤없는(기본형: 없다): 「형용사」 어떤 일이나 현상이나 증상 따위가 생겨 나타나지 않은 상태이다. ◆ ‘없다’는 동
사와 형용사가 있는데 형용사로 거의 99% 쓰인다. ★ 따라서 정답은 형용사로 쓰인 ㉠, ㉣, ㉤를 고르면 된다
```
