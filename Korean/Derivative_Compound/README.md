◆ 파생어(派生語): 실질 형태소에 접사가 결합하여 하나의 단어가 된 말. 명사 ‘부채’에 
접미사 ‘-질’이 붙은 ‘부채질’, 동사 어간 ‘덮-’에 접미사 ‘-개’가 붙은 ‘덮개’, 명사 ‘버선’ 
앞에 접두사 ‘덧-’이 붙은 ‘덧버선’ 따위가 있다. ◆ 합성어(合成語): 둘 이상의 실질 형태소가 결합하여 
하나의 단어가 된 말. ‘집안’, ‘돌
다리’ 따위이다. ① ‘웃음’은 어근 ‘웃-’에 접미사 ‘-음’이 붙어 명사가 된 파생어이다. (O)
 ◆ 웃(실질 형태소) + 음(명사화 접미사) = 파생어
