17. ‘본용언 + 보조 용언’의 구성이 아닌 것은?
① 영수는 쓰레기를 주워서 버렸다. 
② 모르는 사람이 나를 아는 척한다. 
③ 요리 맛이 어떤지 일단 먹어는 본다.
 ④ 우리는 공부를 할수록 더 많은 것을 알아 간다. 
정답 ①
해설 ‘주워서 버리다’는 ‘줍다’와 ‘버리다’의 의미가 
모두 살아 있으므로 본용언 + 본용언의 구성으로 본다. 
오답 풀이 ② 척하다: 앞말이 뜻하는 행동이나 상태를 
거짓으로 그럴듯하게 꾸밈을 나타내는 보조 동사
③ 보다: 어떤 행동을 시험 삼아 함을 나타내는 말을 
뜻하는 보조 동사
④ 가다: 말하는 이, 또는 말하는 이가 정하는 어떤 기준점에서 멀어지면서 앞말이 뜻하는 행동이나 상태가 계속 진행
됨을 나타내는 말을 뜻하는 보조 동사
