# ROM
```java
32K×8비트 ROM 칩에 대한 설명으로 옳지 않은 것은?
① 이 ROM 칩 4개와 디코더(decoder)를 이용하여 128K×8비트 ROM 모듈을 구현할 수 있다.
② 데이터 핀은 8개이다.
③ 워드 크기가 8비트인 컴퓨터 시스템에서만 사용된다.
④ 32,768개의 주소로 이루어진 주소 공간(address space)을 갖게 된다.

∙ 32K * 8비트 이므로 기본적으로 워드 크기가 8비트인 컴퓨터 시스템에서 사용될 수 있지만 
병렬 연결을 통해 16비트, 또는 32비트, 64비트 컴퓨터 시스템에서 사용 가능

```
