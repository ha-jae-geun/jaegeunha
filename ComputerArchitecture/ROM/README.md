# ROM
```java
32K×8비트 ROM 칩에 대한 설명으로 옳지 않은 것은?
① 이 ROM 칩 4개와 디코더(decoder)를 이용하여 128K×8비트 ROM 모듈을 구현할 수 있다.
② 데이터 핀은 8개이다.
③ 워드 크기가 8비트인 컴퓨터 시스템에서만 사용된다.
④ 32,768개의 주소로 이루어진 주소 공간(address space)을 갖게 된다.

∙ 32K * 8비트 이므로 기본적으로 워드 크기가 8비트인 컴퓨터 시스템에서 사용될 수 있지만 
병렬 연결을 통해 16비트, 또는 32비트, 64비트 컴퓨터 시스템에서 사용 가능

```


# 바이오스
```java
하드디스크에 대한 설명으로 옳지 않은 것은?
① 하드디스크는 데이터접근 방식이 직접접근 방식인 보조기억 장치이다.
② 바이오스(BIOS)는 하드디스크에 저장된다.
③ 하드디스크는 주기억장치보다 접근 속도가 느리다.
④ 하드디스크는 전원이 꺼져도 저장된 데이터가 지워지지 않는다.

BIOS
∙ Basic Input/Output System, 기본 입·출력 시스템
∙ 컴퓨터 시스템의 전원을 켜면 운영체제가 동작하기 전에 컴퓨터의 자기 진단과 주변기기 점검 등을 실시하는 프로그램
∙ BIOS는 시스템 초기화를 수행하고, POST(Power On Self Test), 부트스트랩 로더(Bootstrap Loader)를 포함
∙ BIOS는 ROM에 저장되기 때문에 ROM-BIOS라고도 함



3번
· (  ㉠  )에 저장된 바이오스(BIOS)가 실행되어 컴퓨터에 장착된 하드웨어 장치들의 상태를 점검한다.
· (  ㉡  )에 저장되어 있는 운영체제가 (  ㉢  )로/으로 로드(load)된다.
· 운영체제의 실행이 시작된다.


①		㉠ 보조기억장치 ㉡ ROM ㉢ 주기억장치
②		㉠ 보조기억장치 ㉡ 주기억장치 ㉢ ROM
③		㉠ ROM ㉡ 보조기억장치 ㉢ 주기억치
④		㉠ ROM ㉡ 주기억장치 ㉢ 보조기억장치

BIOS
∙ Basic Input/Output System, 기본 입·출력 시스템
∙ 컴퓨터 시스템의 전원을 켜면 운영체제가 동작하기 전에 컴퓨터의 자기 진단과 주변기기 점검 등을 실시하는 프로그램
∙ BIOS는 시스템 초기화를 수행하고, POST(Power On Self Test), 부트스트랩 로더(Bootstrap Loader)를 포함
∙ BIOS는 ROM에 저장되기 때문에 ROM-BIOS라고도 함
 
부트스트랩로더(Bootstrap Loader)
∙ 보조기억장치에 저장되어 있는 운영체제를 찾아 메모리로 로드(Load)하는 역할을 수행

```
