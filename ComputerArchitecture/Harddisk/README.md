# 바이오스
```java
하드디스크에 대한 설명으로 옳지 않은 것은?
① 하드디스크는 데이터접근 방식이 직접접근 방식인 보조기억 장치이다.
② 바이오스(BIOS)는 하드디스크에 저장된다.
③ 하드디스크는 주기억장치보다 접근 속도가 느리다.
④ 하드디스크는 전원이 꺼져도 저장된 데이터가 지워지지 않는다.

BIOS
∙ Basic Input/Output System, 기본 입·출력 시스템
∙ 컴퓨터 시스템의 전원을 켜면 운영체제가 동작하기 전에 컴퓨터의 자기 진단과 주변기기 점검 등을 실시하는 프로그램
∙ BIOS는 시스템 초기화를 수행하고, POST(Power On Self Test), 부트스트랩 로더(Bootstrap Loader)를 포함
∙ BIOS는 ROM에 저장되기 때문에 ROM-BIOS라고도 함
```

# 하드디스크
```java
7. <보기>와 같은 특성을 갖는 하드 디스크의 최대 저
장 용량은?

<보기>
◦ 실린더 (cylinder) 개수: 32,768 개
◦ 면 (surface) 개수: 4개
◦ 트랙 (track) 당 섹터 (sector) 개수: 256 개
◦ 섹터크기 (sector size): 512 bytes
1 4GB 2 16GB 3 64GB 4 1TB

답 2
트랙 당 섹터 수가 256개이고, 섹터 하나의 크기가 512byte 이
므로, 하나의 트랙은 256 × 512 = 128KB 이다.
한 면에 있는 트랙의 수는 실린더의 수와 같다.
그러므로 한 면은
32768 × 128KB = 32K × 128KB = 4096MB = 4GB
총 4면으로 구성되어 있으므로 전체 용량은 16GB가 된다.
○ 하드디스크 용량
= 헤드 수 × 실린더 수 × 섹터 수 × 섹터당 용량
(위의 식에서 헤드 수는 디스크 면 수와 같다.)
```
