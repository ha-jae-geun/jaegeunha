⇨ 먼저 사과한 다음, 응해줄 수 없는 이유를 설명한다. 모호한 태도를 보이는 것보다 단호하게 거절하는 것이 좋다. 단, 정색을 하면서 거절하는 경우 자칫 인간관계까지 나빠질 수 있으므로 주의해야 한다.
