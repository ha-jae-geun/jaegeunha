# 원가, 정가, 할인 판매가
1. x의 이윤 붙인정가 = 원가 * (1+x/100)
2. x의 할인 붙인정가 = 원가 * (1-x/100)

## 문제 1
```java
어떤 문가의 정가는 원가의 1.5배이다. 이 때 20% 할인해서 36000에 판매한다면 원가는 할인 가격보다 ㅇ러마나 싼가
```

## 식
```java
정가: 1.5x
1.5x(1-20/1000) = 36000
```

# 확률
* 조건부 A 확률: AB교집합 / A   =   n(a교B) / n(A)
* 문제에서 %만 나오는 경우 전체 수량을 100 또는 1000과 같이 계산하기 편리한 수로 잡고 문제를 해결한다.

## 문제1
```java
남자가 16명, 여자가 14명이 근무하고 있고, 남자 9명 여자 6명이 안경 쓰고 있었다. 임의로 뽑은 남자가 안경 쓸 확률은?

풀이: -할 때가 부모: P(B) = 15/30  남자일 때: 9/30

팁: 안경 안쓴 사람 제외 -> 그중에서 분수로 만들면 답: 9/15  
```

# 평균
* 도수분포표에서의 평균: (계급값X도수)의 총합/도수의 총합
* 가평균을 이용한 평균: 가평균 + 과부족의 평균
    * 과부족: 변량/계급값 - 가평균
    * 가평균: 가짜 평균
    
```java
10명의 키 평균 구하시오: 164, 181, 175, 178, 159, 180, 167, 172, 171, 183

1. 가평균: 170으로 임의로 잡음
2, 170 + 30(가평균과의 차이)/10
```

## 문제 2
```java
도수 분포표의 평균의 중간값: 55 5명, 65 11명, 75 18명, 85 9명, 95 7명

팁: 가평균 + ((가평균(75) - 55) * 5) 등등)
꿀팁:  1. 가평균을 기준으로 양쪽을 분쇄하고 가평균으로 옮김
      2. 55의 5명과 95의 5명을 가평균으로 보내면 가평균은 *0이니 계산 쉬워짐
```
