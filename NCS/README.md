## 농도
* 농도가 15%인 소금물과 농도가 x%의 소금물을 4:3의 비율로 섞어서 농도가 18%인 소금물을 만드려고 한다.

```java
1. 소금물을 4k, 3k로 지정
2. 15/100*4k + x/100*3k = 18/100*7k

꿀팁: 소금물이 4:3의 비율이면 18%를 기준으로 18-3, 18+4여야 함
답: 22 
```
